#  Simulador de Ordenamiento de Pedidos

##  Descripción

Este proyecto es un **dashboard interactivo** que permite visualizar en tiempo real el funcionamiento de diferentes algoritmos de ordenamiento aplicados a pedidos.

El sistema muestra:

* Representación visual del arreglo (barras)
* Tabla dinámica de pedidos
* Métricas de ejecución
* Código paso a paso (C++ y Python)

---

##  Objetivos

* Comprender algoritmos de ordenamiento de forma visual
* Analizar rendimiento mediante métricas
* Comparar implementaciones en distintos lenguajes

---

##  Algoritmos implementados

* Bubble Sort
* Selection Sort
* Insertion Sort
* Shell Sort
* Quick Sort
* Merge Sort
* Heap Sort

---

##  Funcionalidades

* Generación de datos aleatorios o parcialmente ordenados
* Control de velocidad de simulación
* Visualización paso a paso
* Cambio de lenguaje (C++ / Python)
* Métricas en tiempo real:

  * Comparaciones
  * Intercambios
  * Escrituras
  * Tiempo de ejecución

---

##  Interfaz (Dashboard)

El sistema está diseñado como un panel profesional:

*  Sidebar de control
*  Visualización de datos
*  Tabla de pedidos
*  Código en ejecución
